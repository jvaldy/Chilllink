# Figma Import Assets

These files are designed to be imported directly into Figma.

## Recommended import method

Import the screen SVGs one by one from `screens/`.

Main files:
- `screens/auth-login.svg`
- `screens/auth-register.svg`
- `screens/dashboard-chat.svg`
- `screens/modal-create-workspace.svg`
- `screens/modal-profile.svg`
- `screens/modal-workspace-members.svg`
- `screens/modal-channel-settings.svg`
- `screens/modal-delete-workspace.svg`

Optional:
- `chilllink-mockup-board.svg`

## Import steps

1. Open Figma
2. Create a new design file
3. Drag one file from `screens/` into the canvas
4. Convert the imported SVG into a frame if needed
5. Repeat for the other screens

## Included screens

- Login
- Register
- Chat dashboard
- Create workspace modal
- Profile modal
- Workspace members modal
- Channel settings modal
- Delete workspace confirmation

## Notes

- `chilllink-mockup-board.svg` is only a contact sheet.
- The `screens/` files are the ones intended for real Figma import.
- These are editable SVG imports, not native `.fig` files.
